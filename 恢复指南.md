# 仿写工具恢复指南

## 如果文件损坏了怎么办？

### 方案1：从备份恢复
```bash
# 查看所有备份
ls -lh fangxie_tool.py.backup*

# 恢复最新备份
cp fangxie_tool.py.backup_20260314_XXXXXX fangxie_tool.py
```

### 方案2：从独立协议文件恢复核心逻辑

如果主文件损坏，但核心逻辑文件完好，可以这样恢复：

#### 步骤1：准备干净的基础文件
从git恢复3月4日版本：
```bash
git restore fangxie_tool.py
```

#### 步骤2：导入最新的核心逻辑
打开 `prompt_protocols.py`，复制以下三个变量的内容：
- `title_explosive_protocol` （标题生成协议）
- `style_keywords_protocol` （文风关键词库）
- `explosive_content_protocol` （正文生成协议）

#### 步骤3：替换主文件中的对应部分

**位置1：替换标题生成协议（约3235行）**
找到：
```python
title_positive_protocol = """## 标题风格硬规则（最高优先级）
...
"""
```
替换为：
```python
title_explosive_protocol = """## 爆款标题硬规则（最高优先级）
...（从prompt_protocols.py复制）
"""
```

**位置2：添加文风关键词库（在title_explosive_protocol之后）**
```python
style_keywords_protocol = """## 文风关键词库（增加多样性）
...（从prompt_protocols.py复制）
"""
```

**位置3：添加正文生成协议（在style_keywords_protocol之后）**
```python
explosive_content_protocol = """## 爆款正文硬规则（8大核心规则）
...（从prompt_protocols.py复制）
"""
```

**位置4：修改Prompt引用（约3280行）**
找到：
```python
{hooks_instruction}
{dynamic_instruction}
{strict_output_protocol}
{title_positive_protocol}
{hotspot_alignment_protocol}
{opening_hook_protocol}
```
替换为：
```python
{hooks_instruction}
{dynamic_instruction}
{strict_output_protocol}
{title_explosive_protocol}
{explosive_content_protocol}
{style_keywords_protocol}
{hotspot_alignment_protocol}
{opening_hook_protocol}
```

#### 步骤4：添加去重方法

参考 `去重机制说明.md`，添加两个方法：
1. `check_title_duplicate()` - 在 `get_generated_title_set()` 方法后
2. `check_reference_duplicate()` - 在 `check_title_duplicate()` 方法后

#### 步骤5：调用去重方法

**位置1：参考文章去重（约2247行）**
```python
# 把参考文案追加到素材库（改写前先检查去重）
for article in articles:
    # 检查参考文章是否重复
    if self.check_reference_duplicate(article):
        self.log(f"  跳过重复的参考文案")
        continue
    self.append_reference_to_library(article, flow_type)
```

**位置2：标题去重（约2433行）**
```python
selected_title = random.choice(available_titles)

# 检查标题是否与历史标题重复
if self.check_title_duplicate(selected_title):
    self.log(f"  → 标题重复，尝试选择其他标题")
    # 尝试其他标题
    for alt_title in titles:
        if alt_title != selected_title and not self.check_title_duplicate(alt_title):
            selected_title = alt_title
            self.log(f"  → 改用标题: {selected_title}")
            break
else:
    self.log(f"  → 随机选中: {selected_title}")
```

## 快速检查清单

## 测试方法

恢复后运行测试：
```bash
python fangxie_tool.py
```

如果出现语法错误，检查：
1. 文件编码是否为UTF-8
2. 是否有智能引号（""）而不是普通引号（""）
3. 三引号是否完整
