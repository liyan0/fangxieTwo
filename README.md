# 百家号仿写文案工具

一个带GUI界面的百家号引流文案仿写工具，支持流式/非流式API调用，自动生成高质量仿写文案。

## 功能特点

- 🎯 **智能仿写**：分析参考文案的爆点和结构，生成全新原创内容
- 🎲 **随机开头**：12类150+种开头钩子，每次生成都像开盲盒
- 😊 **风格多样**：强制至少1篇轻松/温暖/反转风格，避免全是压抑内容
- 🔄 **双模型切换**：主模型失败自动切换备用模型
- 📊 **字数保障**：自动检查字数，低于1500字自动重试
- 📝 **多种引流**：支持置顶引流、橱窗引流、带货引流
- 💾 **Word输出**：自动保存为Word文档

## 安装

### 1. 安装Python
需要Python 3.8+，下载地址：https://www.python.org/downloads/

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 运行工具
```bash
python 仿写文案工具.py
```

或者双击 `启动工具.bat`

## 使用说明

### 1. 配置API
首次使用需要配置API：
- 点击「展开API配置」
- 填入你的API地址、Key、模型名称
- 点击「保存配置」

支持的API格式：OpenAI兼容格式（如各种中转站）

### 2. 准备参考文案
- 创建一个txt文件，写入参考文案
- 多篇文案用「参考文案：」分隔

示例：
```
参考文案：
这里是第一篇参考文案的内容...

参考文案：
这里是第二篇参考文案的内容...
```

### 3. 选择引流类型
- **置顶引流**：引导用户去看主页置顶视频
- **橱窗引流**：引导用户去看主页橱窗商品
- **带货引流**：直接推荐商品（需填写商品信息）
- **纯夸赞不引流**：只输出正能量内容，不引流

### 4. 开始生成
- 选择参考文案文件
- 选择输出保存路径
- 点击「开始生成」
- 等待生成完成，自动保存Word文档

## 配置文件说明

配置保存在 `fangxie_config.json`，包含：
- API配置（URL、Key、模型）
- 流式/非流式设置
- 引流话术模板库

## 常见问题

### Q: 提示缺少模块？
A: 运行 `pip install -r requirements.txt` 安装依赖

### Q: API调用失败？
A: 检查API配置是否正确，网络是否通畅

### Q: 生成的文案字数不够？
A: 工具会自动重试，最多4次。如果还是不够，可能是API返回限制

### Q: 如何添加自定义引流话术？
A: 在引流话术区域输入内容，点击「保存当前话术」
