# 配音神器自动化脚本开发记录

## 项目目标
通过比特浏览器自动化操作配音神器网站(https://peiyinshenqi.com/tts/index)，实现批量文案配音生成。

## 当前进度

### 已完成功能
1. **打开比特浏览器** - 通过BitBrowser API打开序号11的浏览器
2. **清空文本框** - 点击清空按钮，处理"立即清除"弹窗
3. **输入文案** - 点击编辑区域(contenteditable div)，等待2秒后输入文本
4. **选择音色** - 点击"云泽-通用"音色
5. **选择语速** - 点击语速下拉框，选择"1.2倍慢速"
6. **选择情感强度** - 选择"2倍"情感
7. **点击合成配音** - 点击合成配音按钮
8. **点击开始合成** - 处理弹窗，点击"开始合成"按钮
9. **等待合成完成** - 监测loading遮罩消失（el-loading-mask is-fullscreen）
10. **下载配音** - 点击下载配音按钮
11. **下载字幕** - 点击下载字幕按钮，等待loading消失

### 待解决问题
**保存对话框操作失败**：点击下载配音/字幕后会弹出保存对话框，需要：
1. 在文件名输入框中输入文件名（如"我是一个小笨蛋"）
2. 点击保存按钮

当前使用pyautogui的Ctrl+A、Ctrl+V、Enter操作没有生效，可能原因：
- 焦点没有在对话框的文件名输入框上
- 需要先点击输入框获取焦点
- 可能需要使用其他方式定位和操作对话框
